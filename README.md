git clone <repository>  -> cloning dari remote server ke komputer (mirip proses download)

ls -> melihat daftar file dan folder

git status -> melihat perubahan file yang telah dilalukan, untuk memastikan hanya perubahan tertentu saja yang diupload

git add <file yang akan dicommit> -> menambahkan file yang akan dicommit (commit = komitmen)

git commit -m <pesan commit> -> melakukan komitmen terhadap perubahan (finalisasi)

git push origin main -> mentransfer perubahan ke remote server (github/gitlab/dsb.)
